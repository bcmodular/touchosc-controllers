# touchosc-controllers

Collection of MIDI/OSC controllers made using TouchOSC.

## Projects

| Project | Description |
|---------|-------------|
| [**sp404-mk2**](sp404-mk2/README.md) | Roland SP-404 MKII effects controller — 5 buses, 46 FX, presets/scenes, BCR2000 + Launchpad Pro |
| [p6-granular](p6-granular/) | BCR2000 layout for P6 granular |
| [s-1](s-1/) | Roland S-1 synth layout |
| [utils](utils/) | MIDI tester layouts and Launchkey Mini scripts |

Shared build tooling: `python3 tools/toscbuild.py build <project-dir>` (see [docs/tosc-format.md](docs/tosc-format.md)).
